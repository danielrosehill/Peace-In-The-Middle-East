# Actors

The experiment/model will attempt to capture the breadth of actors involved in shaping the geopolitical reality of the modern Middle East. The experiment will attempt to be somewhat nuanced although achieving any meaningful degree of nuance is impossible in a setup of this nature.

---

## State Delegation Structure

Each state delegation comprises a full apparatus of government. Not all roles apply to all states (e.g., some have presidents, others monarchs, some both).

### Standard Delegation Composition

| Role | Function | Chamber Access |
|------|----------|----------------|
| **Head of State** | President / Monarch / Supreme Leader | Plenary only (high-level) |
| **Head of Government** | Prime Minister / Chancellor | Plenary, Government Forum |
| **Foreign Minister** | Chief diplomat, MFA head | All chambers |
| **Defense Minister** | Civilian military oversight | Government Forum, Security track |
| **Military Chief** | Head of armed forces / Chief of Staff | Security track, crisis sessions |
| **Intelligence Chief** | Head of primary intelligence agency | Security track (limited) |
| **Finance Minister** | Economic policy | Economic track |
| **Justice Minister / Attorney General** | Legal framework, rule of law | Normalization track |
| **Chief Justice** | Head of judiciary (observer capacity) | Framework negotiations |
| **Diplomatic Corps Lead** | Senior ambassador / Permanent Rep | All chambers |
| **Parliamentary Leader** | Speaker / Majority leader | People-to-People, Normalization |
